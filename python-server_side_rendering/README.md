## Python - Server-Side Rendering

This project is about learning to use Server-Side Rendering.

* [data/](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/data), the directory containing data files for the Flask application.

* [templates/](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/templates), the directory containing template files for the Flask application.

* [task_00_intro.py](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/task_00_intro.py), a function generating invitations from template and attendees.

* [task_01_jinja.py](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/task_01_jinja.py), a simple Flask application using Jinja for template rendering.

* [task_02_logic.py](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/task_02_logic.py), a simple Flask application reading items from serialized data in JSON file.

* [task_03_files.py](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/task_03_files.py), a simple Flask application reading products from serialized data in JSON and CSV files.

* [task_04_db.py](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering/task_04_db.py), a simple Flask application reading products from a database using SQLite.

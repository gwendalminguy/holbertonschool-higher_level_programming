# Higher Level Programming

This repository focuses on high level programming languages.

* [Python - Hello World](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-hello_world), a project to learn about using Python basics such as formatted strings, concatenation and slicing.

* [Python - If/Else, Loops & Functions](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-if_else_loops_functions), a project to learn about using Python basics such as conditions, loops and functions.

* [Python - Import & Modules](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-import_modules), a project to learn about imports and modules in Python.

* [Python - Data Structures (Lists, Tuples)](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-data_structures), a project to learn about using Python data structures such as lists and tuples.

* [Python - More Data Structures (Sets, Dictionaries)](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-more_data_structures), a project to learn about using more Python data structures such as sets and dictionaries.

* [Python - Exceptions](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-exceptions), a project to learn about handling exceptions and errors in Python.

* [Python - Test-Driven Development](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-test_driven_development), a project to learn about creating tests and writing code documentation.

* [Python - Classes & Objects](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-classes), a project to learn about using classes and objects in Python.

* [Python - More Classes & Objects](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-more_classes), a project to learn about using more classes and objects in Python.

* [Python - Inheritance](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-inheritance), a project to learn about using inheritance.

* [Python - Abstract Classes & Interfaces](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-abc), a project to learn about using abstract classes and interfaces in Python.

* [Python - Input/Output](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-input_output), a project to learn about opening, reading and writing files in Python, and using the JSON format.

* [Python - Serialization](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-serialization), a project to learn about serializing and deserializing data.

* [RESTful API](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/restful-api), a project to learn about creating a RESTful APIs.

* [SQL - Introduction](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/SQL_introduction), a project to learn about using SQL basics with MySQL.

* [SQL - More Queries](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/SQL_more_queries), a project to learn about using more SQL queries with MySQL.

* [Python - Object-Relational Mapping](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-object_relational_mapping), a project to learn about using ORMs with MySQLdb and SQLAlchemy.

* [JavaScript - Warm Up](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/javascript-warm_up), a project to learn about using JavaScript basics.

* [JavaScript - DOM Manipulation](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/javascript-dom_manipulation), a project to learn about manipulating the Document Object Model (DOM).

* [Python - Server-Side Rendering](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-server_side_rendering), a project to learn about using server-side rendering in Python.

* [Python - Everything Is Object](https://github.com/gwendalminguy/holbertonschool-higher_level_programming/tree/main/python-everything_is_object), a project to learn about types and  mutability in Python.
